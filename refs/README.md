# References

This directory contains references needed to work with TPOF. Some references are tracked by the repository, while others are ignored and must be provided by the developer. 

## `rdn`

Contains the Relic Developers Network tools.

## `homeworld2-big`

Contains the decompiled `homeworld2.big` file, with a LuaDC pass on the compiled Lua files.

## `english-big`

Contains the English locale information; decompiled from `english.big`.
